# WebUtilsPro

**WebUtilsPro** (webutilspro.com) is a dynamic, all-in-one online workspace packed with free browser-based developer, data, design, AI, security, and text utilities. Everything runs client-side wherever possible, so your code and data stay private and never leave your device.

## 🚀 Featured Tools

- **Cloud Lab** – Test AWS-style S3, SQS, SNS, Lambda, DynamoDB, API Gateway, and webhook flows.
- **Webhook Inspector** – Capture and debug incoming HTTP requests in real time with zero configuration.
- **API Playground** – Professional REST API testing with code generation and request history.
- **MCP Tool Tester** – Test and debug Model Context Protocol servers in a live sandbox.
- **OTLP Endpoint Inspector** – Capture OpenTelemetry exports and inspect decoded spans, resources, and attributes.
- **Visual DB Architect** – Design database schemas visually with a draggable interactive canvas.

## 🛠️ Developer Utilities

- **JavaScript Compiler** – Run, test, and debug JavaScript online with console output, syntax validation, runtime errors, and debug traces (executes safely inside a browser Web Worker; nothing is uploaded to servers).
- **Code Formatter** – Beautify JavaScript, HTML, CSS, JSON, and SQL instantly, powered by Prettier.
- **JSON Validator & Formatter** – Validate JSON syntax, format with indentation, minify for production, and get detailed line-numbered error messages.
- **JSON ↔ CSV Converter** – Bidirectional conversion between JSON and CSV formats with validation, download, and copy support.
- **Regex Tester** – Test and debug regular expressions with real-time matching and highlighting.
- **Markdown to HTML** – Convert Markdown to HTML with live preview and syntax highlighting.

## 🤖 AI-Powered Tools

- **AI Error Explainer** – Get AI-powered explanations and fixes for programming errors.
- **Prompt Optimizer** – Optimize AI prompts for clarity and better results.
- **PDF Text Extractor** – Extract text from complex PDFs containing images with AI.
- **Transcript Generator** – Convert speech to text instantly in your browser.

## 🔐 Security & Data Tools

- **JWT Decoder** – Decode JWT tokens and generate cryptographic hashes.
- **Base64 Encoder/Decoder** – Encode and decode Base64 strings for data transmission.
- **URL Encoder/Decoder** – Convert URLs to safe encoded format and back.
- **Password Generator** – Create secure passwords and check password strength.
- **UUID Generator** – Generate unique identifiers instantly.

## 🎨 Design & Text Tools

- **Color Picker** – Pick colors, convert formats, and generate CSS gradients.
- **QR Code Generator** – Generate custom QR codes for URLs, text, and more.
- **Text Tools** – Count characters and words, convert case, and clean up text.
- **Lorem Ipsum Generator** – Generate placeholder text for designs and mockups.
